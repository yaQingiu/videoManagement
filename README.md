# Vue 3 + Vite
This is a single, elegant video management website. It contains following features:
1 Using email validation to register, and using email information as an unique ID.
2 Using lazy router to minimize loading time.
3 Checking token consistency before switching pages. 
3 Using Pinia to manage global user & video & category information, reducing web & server interaction time.
4 Using SSE to implement realtime conversation function that likes daily chatting.
5 Upload large video files using the chunked upload function, with a progress bar displayed at the top and support for resumable uploads.
